# Prompt 7: Application Use Cases

Context:
Implement the business logic in the Application layer.
Constraints:
- No UI or database direct reference.
- Interfaces for use cases (e.g., IRegisterRestaurantUseCase, IRegisterOrderUseCase).
- Input/Output DTOs.
- Use repository interfaces via DI.
Output:
UseCase interfaces, DTOs, and Service implementations.
